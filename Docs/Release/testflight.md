# Internal TestFlight Release Notes

## Release Target

Quiet Term beta is internal TestFlight only.

Public App Store launch, pricing, and purchase flows are out of scope for the first beta.

## Release Candidate Checklist

- CI build and tests pass.
- SSH fixture suite passes for supported auth and host-key scenarios.
- Five beta readiness journeys pass.
- Privacy and encryption/export answers are reviewed.
- OSS notices are current.
- Diagnostics redaction tests pass.
- No known P0/P1 beta blockers remain open.

## Tester Focus

Ask internal testers to focus on:
- Creating and editing host profiles.
- Connecting with password, key, passphrase, and MFA-style prompts.
- Host-key first-use and changed-key warnings.
- Multiple foreground tabs.
- Hardware keyboard and software keyboard input.
- Diagnostics export after failures.
